🎯 Project goal: A bite‑size analytics pipeline that shows function‑level dependencies across four modules.

📚 Project Modules:

  • 🚀 pipeline.py – entry‑point, orchestrates everything.
  • 🔧 utils.py – helpers: log, load_csv, save_json.
  • 🧮 math_ops.py – maths: square, mean.
  • ⚙️ processor.py – clean & feature‑engineer: clean_row → enrich_row → process_rows; stats via compute_stats.

🔄 Execution flow:

  1. run() loads the CSV via load_csv 📂
  2. Rows go through process_rows → clean_row → enrich_row (uses square & log) 🔄
  3. Then compute_stats summarizes, and save_json writes the outputs 📊
